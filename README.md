# Russell Dudek - Enterprise AI Candidate Campaign

Independent candidate materials for the Member of Technical Staff, Enterprise AI opportunity represented in the job posting by Quik Hire Staffing.

## Candidate thesis

Enterprise AI scales when every model sits on an operable backplane. Outcome, context, model behavior, runtime integration, human authority, and evidence must share one production contract.

## Public artifact set

- `index.html` - candidate vision and interactive production fault drill
- `resume.html` - exactly two-page role-aligned resume
- `cover-letter.html` - exactly one-page cover letter
- `interview-brief.html` - interview thesis brief
- `90-day-plan.html` - first 90 days
- `backplane-review.html` - one-page Enterprise AI Backplane Review
- `docs/` - generated PDFs for every printable artifact
- `brand-intelligence.md`, `brand-tokens.css`, and `assets/brand/` - identity provenance and implementation
- `campaign-audit.md` and `artifact-manifest.json` - validation record
